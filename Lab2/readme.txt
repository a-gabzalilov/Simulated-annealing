Параметры аргументной строки:
config_file.txt chunk_size
где config_file.txt - конфиг файл, а chunk_size - размер интервалов.

В конфиг файл записаны: 
- Input file
- Output file
- Log file

Структура Input file:
Первая строка - размер N
Вторая строка - через пробел записаны N элементов ряда t
Третья строка - через пробел записаны N элементов ряда Х(t)

Структура Output file:
Первая строка - интервал,
Затем "Degree" - степень полинома,"gammas" - оценка коэффициентов гамма, записанные через пробел, "betas" - оценка коэффициентов бета_1, найденные через гамма, записанные через пробел.


Также дополнительно генерируется файл "annealing.txt":
Записываются значения бета_0 и бета_1 найденные с помощью метода отжига Больцмана и Коши.
